# 反応解釈ルール

> この資料は仮文章であり、本番前に正式情報へ差し替える。


- 滞在時間が伸びる: 興味継続の可能性。
- BBox幅が大きくなる: 近づいた可能性。
- 人数が増える: 周囲を巻き込めた可能性。
- 人数が減る: 興味低下または通行の可能性。
- leaving_risk高: 短く締める/引き止める。
- smile推定: 反応良好の可能性。
- neutral推定: 悪いとは限らない。
- expression_confidence低: 表情判断を弱く扱う。
- 戻ってきた: 興味維持の可能性。
- 属性情報は演出に使えても、攻撃的・差別的に使わない。
